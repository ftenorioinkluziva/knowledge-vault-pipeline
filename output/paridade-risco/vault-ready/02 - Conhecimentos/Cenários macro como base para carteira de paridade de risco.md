# Cenários macro como base para carteira de paridade de risco

> **Resumo:** A leitura dos quatro cenários serve como preparação para montar uma carteira de paridade de risco. O próximo passo é ponderar os ativos conforme o regime macro em vez de escolher apenas vencedores isolados.

## Explicação Detalhada

O trecho explica que a identificação dos ativos mais favoráveis em cada cenário é um passo anterior à construção de uma carteira de paridade de risco. A ideia é mapear quais classes se comportam melhor em cada regime macroeconômico brasileiro e, depois, combiná-las em uma estrutura de alocação mais equilibrada. Assim, o processo passa de uma leitura histórica de desempenho para uma organização sistemática da carteira por cenário.

## Aplicação Prática / Protocolo

Usar o mapa de cenários como insumo para a etapa seguinte de ponderação e diversificação entre classes de ativos. Risco: montar carteira apenas com base no cenário mais recente, sem considerar a possibilidade de transição entre regimes.
